The replication package contains the following:-

1. Java files that compute the developer interaction network, for each dataset.
2. Java files that fetch the relevant data, for each dataset.
3. R files that perform the causal analysis for each dataset.

Dataset links:

Android:  The Android dataset was obtained from the repository provided by Shihab et al. in their paper.
Full Citation: Shihab, E., Kamei, Y., Bhattacharya, P.: Mining challenge 2012: The Android platform. In: 2012 9th IEEE Working Conference on Mining Software Repositories
(MSR), pp. 112–115 (2012). https://doi.org/10.1109/MSR.2012.6224307 .ISSN: 2160-1860

Eclipse and OpenStack: The OpenStack and Eclipse datasets were obtained from the database dumps provided in the research paper by Gonzalez-Barahona et al.
